# Basic Informational Site

A simple Node.js web server that serves static HTML pages based on URL routing, built as part of [The Odin Project](https://www.theodinproject.com/).

## Skills Demonstrated

- Setting up a basic web server using Node.js and the HTTP module
- Handling HTTP requests and implementing URL-based routing
- Reading files from the filesystem using the `fs` module
- Understanding the request/response cycle in server-side JavaScript
- Working with built-in Node.js modules (http, fs)
- Serving static files
